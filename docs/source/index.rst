F.R.O.T.H.
==========

.. figure:: /_static/images/rpg-image-1.png

   art Â© `Akim Kaliberda <https://www.artstation.com/artwork/NzDk5>`_

F.R.O.T.H. stands for **"Fantasy Rules of the House"**. It means any TTRPG hack based on classic and/or traditional rules systems.

This website hosts **MY** FROTH systems and procedures, but anyone can create their own FROTH game based on these or already existing rules systems.
